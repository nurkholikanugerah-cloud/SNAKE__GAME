# Deskripsi Projek
Jadi projek yang akan kami buat adalah projek game ular (Snake) yang dimana ular digambarkan sebagai titik berwarna hijau yang bergerak secara terus-menerus. Tugas seorang pemain adalah mengarahkan ular tersebut untuk memakan makanan berbentuk kotak merah yang muncul secara acak di dalam arena. Setiap kali makanan dimakan, skor pemain akan bertambah dan tubuh ular akan semakin panjang. Seiring bertambahnya panjang ular, tingkat kesulitan permainan juga meningkat. Pemain harus berhati-hati agar ular tidak menabrak dinding arena atau menyentuh tubuhnya sendiri, karena hal tersebut akan menyebabkan permainan berakhir (game over).

# Yang Dibutuhkan Oleh Projek ini
Yang akan digunakan dalam projek ini adalah Visual Studio Code. Di dalam VSC kami menggunakan HTML, CSS, dan JAVASCRIPT

# How To Use?
- Game langsung berjalan saat dibuka
- Gerakkan ular dengan tombol panah keyboard atau tombol arah di layar
- Arahkan ular ke makanan untuk menambah skor dan panjang tubuh
- Hindari dinding dan tubuh ular sendiri
- Jika menabrak maka Game Over
